## Description
nestjs와 mssageQueue(rabbitMQ)를 활용하여
mysql과 mongodb로 CRUD를 적용하는 프로젝트


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
# package.json listen 스크립트 추가
$ npm run listen
```

## Add package
1. --save @nestjs/typeorm typeorm mysql
    * nestjs에서 relration ORM을 사용하기 위해 설치

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
